    About Merge PDF

    Select multiple PDF files and combine them into a single document. This is useful when a document was split across multiple pages and needs to be recombined.

    Files are merged in the order displayed. Drag to reorder before merging.

    Tips

    • The merged file downloads immediately.
    • Use Split PDF first to break apart a combined document.
